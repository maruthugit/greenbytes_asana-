# GreenBytes Asana — Quick Training (1 Page)

Use this as a simple daily guide for staff.

---

## 1) Login
- Open the app URL → login screen
- Enter your email + password
- After login, you will only see menus you are allowed to use

If you see “403 Forbidden”, ask an Admin to grant access.

---

## 2) Daily Workflow (Recommended)

### A) Check “My Tasks”
- Go to **Dashboard** (home)
- Use tabs:
  - **Upcoming**: what to do next
  - **Overdue**: late items
  - **Completed**: finished work

### B) Create a task
- From **Home**: click **+ Create task** (under **My tasks**)
- Or go to **Tasks**
- Fill:
  - Project
  - Task title
  - Status (Todo / Doing / Done)
  - Due date (optional)
  - Assignee (optional)
  - Task image (optional)
- Click **Add task**

### C) Move tasks on the Board
- Open a project board
- Drag a task card to **Todo / Doing / Done**

### D) Comment on a task
- Go to **Tasks**
- Click a task title to open it
- Type your comment → **Post comment**

---

## 3) Task Status Rules (Simple)
- **Todo**: not started
- **Doing**: in progress
- **Done**: finished

---

## 4) Tips
- Add a due date for better tracking
- Use comments for updates instead of separate messages
- Upload an image if the task needs a reference screenshot/photo

---

## 5) Help
- Missing a menu item? You likely don’t have permission.
- Contact the Admin to enable access.
